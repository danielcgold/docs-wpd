### <span>Templates that has been emptied</span>

For depreciation.

The [Compatibility\_Section template](http://docs.webplatform.org/wiki/Template:Compatibility_Section) has been purposefully muted so we can add the compat data only where we want.

See list of pages [using Template:Compatibility\_Section in an hardcoded fashion](http://docs.webplatform.org/w/index.php?title=Special%3AWhatLinksHere&target=Template%3ACompatibility+Section&namespace=0).

The [Template:Compatibility\_Form\_Section](http://docs.webplatform.org/wiki/Template:Compatibility_Form_Section) should be removed from all templates as they are being replaced with the new [Compatibility template](http://docs.webplatform.org/wiki/Template:Compatibility)
