<h2><span class="mw-headline" id="Notes">Notes</span></h2>
<h2><span class="mw-headline" id="Syntax_2">Syntax</span></h2>
<pre>font-size: <i>absolute-size</i> | <i>relative-size</i> | <i>length</i> | <i>percentage</i>
</pre>
<h2><span class="mw-headline" id="values_2">values</span></h2>
<ul><li><i>absolute-size</i>: A set of keywords indicating predefined font sizes that scale according to font setting preferences or the browser's default values. From small to large, possible values are:
<ul><li><b>xx-small</b></li>
<li><b>x-small</b></li>
<li><b>small</b></li>
<li><b>medium</b></li>
<li><b>large</b></li>
<li><b>x-large</b></li>
<li><b>xx-large</b></li></ul></li></ul>
<ul><li><i>relative-size</i>: </li></ul>
<p>A set of keywords interpreted relative to the parent element's <b>font-size</b>. Possible values are:
</p>
<ul><li><ul><li><b>smaller</b></li>
<li><b>larger</b></li></ul></li></ul>
<ul><li><i>length</i>: A positive numeric value followed by a string designating absolute units (<b>cm</b>, <b>mm</b>, <b>in</b>, <b>pt</b>, <b>pc</b>) or relative units (<b>em</b>, <b>ex</b>, <b>px</b>). If unspecified, the default unit is <b>px</b>. Proportional <b>em</b> measuements are based on the parent element's <b>font-size</b>.</li></ul>
<ul><li><i>percentage</i>: A positive integer followed by a percent (<b>%</b>), indicating the proportion of the parent element's <b>font-size</b>.</li></ul>
